Introduction to TypeScript

What is TypeScript?

TypeScript is a typed superset of JavaScript that compiles down to plain JavaScript. It adds optional static types to JavaScript, enabling better code quality, tooling, and scalability.

Why Use TypeScript?

  • Static Typing: TypeScript catches errors at compile time rather than at runtime.
  • Tooling: Improved IDE support with autocompletion and type checking.
  • Scalability: Especially useful for large-scale applications, enforcing code consistency and maintainability.

Core Features of TypeScript

1. Type Annotations

In TypeScript, you can explicitly specify types for variables, function parameters, and return types.


        let isDone: boolean = false;
        let count: number = 42;
        let name: string = "John";
        
        // Arrays and Tuples
        let numbers: number[] = [1, 2, 3];
        let tuple: [string, number] = ["hello", 10];

2. Enums

Enums allow you to define a set of named constants. They are especially useful when dealing with sets of related values.


        enum Direction {
            Up,
            Down,
            Left,
            Right
        }
        let dir: Direction = Direction.Up;

3. Functions and Return Types

In TypeScript, you can define the types for function parameters and return values.


        function add(x: number, y: number): number {
            return x + y;
        }
        
        // Optional parameters and default values
        function greet(name: string, greeting?: string): string {
            return `${greeting || "Hello"}, ${name}!`;
        }

4. Type Inference

TypeScript can infer types based on the initial value assigned, meaning you don't always need to explicitly declare types.


        let message = "Hello!";  // inferred as string
        let count = 100;         // inferred as number

5. Union Types and Type Aliases

Union types allow a variable to hold more than one type, and type aliases can be used to give a type a name.


        // Union type
        let id: string | number;
        id = "abc";  // valid
        id = 123;    // also valid
        
        // Type alias
        type UserID = string | number;
        let userID: UserID = 101;

6. Interfaces

Interfaces describe the shape of an object, helping enforce structure across your codebase.


        interface User {
            name: string;
            age?: number;  // optional property
        }
        
        let user: User = {
            name: "Alice",
            age: 25
        };
        
        // Optional properties example:
        let anotherUser: User = {
            name: "Bob"  // age is optional here
        };

7. Classes and Access Modifiers

TypeScript supports object-oriented programming with classes, inheritance, and access modifiers such as `public`, `private`, and `protected`.


        class Animal {
            private name: string;
            
            constructor(name: string) {
                this.name = name;
            }
        
            public move(distance: number): void {
                console.log(`${this.name} moved ${distance} meters.`);
            }
        }
        
        class Dog extends Animal {
            public bark(): void {
                console.log("Woof! Woof!");
            }
        }
        
        let dog = new Dog("Rex");
        dog.bark();
        dog.move(10);

8. Generics

Generics allow you to create reusable components that work with a variety of data types.


        function identity(arg: T): T {
            return arg;
        }
        
        let num = identity(10);  // T is number
        let str = identity("hello");  // T is string

9. Modules (Import/Export)

TypeScript supports modular code through the use of `import` and `export`, allowing you to split code into reusable components.


        // math.ts
        export function add(x: number, y: number): number {
            return x + y;
        }
        
        // app.ts
        import { add } from './math';
        
        console.log(add(5, 3));

Setting Up TypeScript

To install TypeScript globally, run:


        npm install -g typescript

Then create a TypeScript file and compile it to JavaScript using the command:


        tsc myfile.ts
